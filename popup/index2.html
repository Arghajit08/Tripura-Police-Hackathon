Problem:

Police Department needs to know instantly the stock positions of articles in the store so that distribution as well as purchase of store items require less time.

Outcome:

One client server based multi log in web based software in which we will get the list of items and its stock position. Search option should also be there. Item add option will be there. In case of distribution, stock position will decrease automatically and vice versa. Report generation is also required to be present.

Major Module:

Admin Console to manage user delegation, report management, User console with stock entry, stock release request and stock disburse. Reporting module of every activity and stock. Printable stock requisition and release document generation. Mobile Scannable reporting and quick response code enabled requisition and disburse slip.

Platform :

Desktop-Open Source Application Mobile- Android but not limited.
